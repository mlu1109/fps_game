All content in this pack was hand made using Gimp, Blender, and Crazybump
PBR Maps generated from base materials are made to be blended

Textures:
Those within the Hand Painted folder are cartoony, not realistic.
Those within the PBR section are made to be realistic.
Those within the PBR Maps section are Maps generated from the Hand Painted
section, repurposed for realistic detail. Color maps must be added to these.

Some include extra maps for optional adjustments in color and style. 
The grass textures will almost always come with "flow" maps, which
can be panned accross the material to simulate movement.
